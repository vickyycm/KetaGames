# 🎮 KetaGames – Plataforma Web de Juegos Interactivos

KetaGames es una plataforma web desarrollada con **Python y Flask**, basada en juegos interactivos de palabras y lógica, integrando **MongoDB** para la persistencia de datos y el uso de **APIs e inteligencia artificial** para personalizar la experiencia del usuario.

La plataforma reúne tres juegos principales:

- 🟨 Palabra del Día  
- 🔵 Rosco  
- 🧩 Contexto    

Antes de comenzar cada juego, el usuario puede ingresar una categoría o temática. A partir de esa información, un modelo de IA adapta dinámicamente el contenido del juego, seleccionando palabras y conceptos relacionados con la temática elegida.

---

## 🎯 Objetivo del proyecto

El objetivo principal es desarrollar una plataforma dinámica e interactiva que combine:

- Desarrollo web con Flask
- Persistencia de datos con MongoDB
- Consumo de APIs
- Generación dinámica de contenido mediante IA
- Arquitectura modular en Python

---

## 🧠 Descripción funcional

El sistema permite a los usuarios interactuar con los tres juegos del sitio: **Palabra del Día**, **Contexto** y **Rosco**, ingresando previamente una palabra o temática que determinará el contenido del juego.

Mediante un modelo de inteligencia artificial, el sistema selecciona automáticamente palabras, conceptos y relaciones asociadas a la temática elegida por el usuario, adaptando la experiencia de juego de forma dinámica.

---

## 🎮 Juegos incluidos

### 🟨 Palabra del Día
Juego basado en descubrir una palabra relacionada con la temática seleccionada.

### 🔵 Rosco
Juego de preguntas y respuestas inspirado en el clásico formato de rosco alfabético.

### 🧩 Contexto
Juego de asociación semántica donde el usuario debe encontrar la palabra objetivo mediante pistas contextuales.
